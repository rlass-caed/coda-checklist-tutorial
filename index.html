📚 TUTORIAL DE UTILIZAÇÃO - CODA

🎯 Objetivo

Este tutorial tem como objetivo padronizar o uso do CODA entre todas as equipes, garantindo que todos tenham o mesmo conhecimento sobre cadastro de checklists, manutenção e acesso às informações.

SEÇÃO 1

📋 COMO CADASTRAR UM CHECKLIST

⚠️
IMPORTANTE: Esta área permite cadastrar tópicos(checklists) necessários para realizar uma atividade existente e já mapeada. Para adicionar atividades novas que ainda não foram mapeadas, procure a supervisão para orientação sobre o processo de mapeamento inicial.
Acessar o Coda - Operação de Dados

Faça login no sistema CODA utilizando suas credenciais.

💡 Exemplo Visual - Tela de Login

📷 Imagem será adicionada aqui

Screenshot da tela de login do CODA

Entrar na Área "Operação de Dados"

Na lista de documentos, clique em "Operação de Dados" para acessar a área de trabalho principal.

💡 Exemplo Visual - Acessar Operação de Dados

Clique no documento "Operação de Dados" na lista de documentos disponíveis.

Navegar até Templates

No painel de navegação esquerdo, siga o caminho hierárquico:

📁 Bases Compartilhadas → 📋 Checklist → 📝 Templates

Esta é a área principal para gerenciamento de todos os modelos de checklist, onde você poderá visualizar, criar e modificar templates existentes.

💡 Exemplo Visual - Navegação

A imagem mostra como navegar pela estrutura de pastas até chegar na seção Templates, destacando cada etapa do caminho.

🔍 Localização: A seção "Bases Compartilhadas" está localizada no menu lateral esquerdo da aplicação.
Adicionar Novo Template

Clique no botão "Adicionar Template" para iniciar o cadastro.

💡 Exemplo Visual - Botão Adicionar
Preencher Campos Obrigatórios

Para cada atividade a ser cadastrada, preencha os seguintes campos:

🔸 Atividade(selecione uma atividade ja existente)
🔸 Tópico
🔸 Descrição
💡 Exemplo Visual - Formulário
📋 Estrutura Hierárquica: As atividades são organizadas em TÓPICOS que servem como agrupadores e que reúnem itens relacionados. Cada TÓPICO funciona como uma categoria que agrupa vários ITENS do checklist, facilitando a organização e execução sequencial das tarefas.
Adicionar Atividade Nova(Outro método)

Existe outra maneira de adicionar itens novos diretamente no template. Este método é útil quando você precisa criar rapidamente um novo tópico ou item dentro de uma atividade existente.

🔸 Localizar Equipe: Encontre a equipe onde deseja incluir a nova atividade
🔸 Expandir Seção: Clique para expandir e visualizar as atividades da equipe selecionada
🔸 Selecionar Atividade: Escolha a atividade desejada e expanda para ver seus tópicos e itens
🔸 Adicionar Novo Item: Defina o tópico que melhor se encaixa o novo item, adicione o título e complete com a descrição detalhada
💡 Exemplo Visual - Método Alternativo
💡 Dica: Este método é mais rápido para adicionar múltiplos tópicos na mesma atividade, pois você não precisa repetir a seleção da atividade a cada novo item.
Reordenar e Excluir Atividades

Após adicionar suas atividades, você pode reorganizar a ordem de exibição ou remover itens que não são mais necessários.

🔸 Ordenação Automática: O sistema numera automaticamente as atividades conforme são adicionadas
🔸 Reordenação Manual: Clique e arraste qualquer atividade com o mouse para a posição desejada
🔸 Atualização Automática: Os números de ordem se ajustam automaticamente após o movimento
🔸 Exclusão de Atividades: Clique com o botão direito na linha e escolha a opção "delete row"
🔸 Confirmar Exclusão: Confirme a ação quando solicitado pelo sistema
💡 Exemplo Visual - Reordenação e Exclusão

Vídeo demonstrando como reordenar atividades arrastando com o mouse e como excluir itens usando o menu contextual

SEÇÃO 2

🔧 MANUTENÇÃO DO CHECKLIST

Realizar Manutenção de Checklists Existentes

Para realizar manutenção em checklists já cadastrados, siga os mesmos passos da SEÇÃO 1 - COMO CADASTRAR UM CHECKLIST até chegar no template. A partir desse ponto, siga os passos específicos de manutenção abaixo:

🔸 Localizar Equipe: Encontre a equipe onde deseja fazer a manutenção
🔸 Expandir Seção: Clique para expandir e visualizar as atividades da equipe selecionada
🔸 Selecionar Atividade: Escolha a atividade desejada e expanda para ver seus tópicos e itens
🔸 Realizar Manutenção: Adicionar, editar ou excluir itens e tópicos conforme necessário
💡 Exemplo Visual - Processo de Manutenção
💡 Dica: Para adicionar novos itens, use os mesmos procedimentos explicados na Seção 1. Para editar, clique diretamente no campo desejado. Para excluir, use o botão direito na linha e selecione "delete row".
⚠️ Importante: Lembre-se que os TÓPICOS funcionam como AGRUPADORES de itens relacionados. Mantenha essa organização durante a manutenção.
SEÇÃO 3

🚀 COMO UTILIZAR O CHECKLIST

Entrar na Área "Operação de Dados"

Na lista de documentos, clique em "Operação de Dados" para acessar a área de trabalho principal.

💡 Exemplo Visual - Acessar Operação de Dados

Clique no documento "Operação de Dados" na lista de documentos disponíveis.

Escolher Sua Equipe

Na tela inicial, selecione sua equipe na lista de equipes disponíveis.

Nesta tela você pode ver como selecionar sua equipe na lista disponível.

Acessar Painel da Equipe

Entre no painel da sua equipe onde você poderá visualizar todos os projetos cadastrados.

💡 Exemplo Visual - Painel da Equipe

Nesta tela você seleciona o painel da equipe que deseja

Selecionar Projeto

Escolha o projeto desejado e clique no botão verde "Abrir" para acessá-lo.

💡 Exemplo Visual - Seleção de Projeto
🔍 Dica de Pesquisa: É possível pesquisar pelo código do projeto usando o campo de pesquisa localizado na parte direita da tela. Isso facilita encontrar projetos específicos em listas extensas.
📋 Projetos Disponíveis: Todos os projetos exibidos aqui já existem no sistema. Se não encontrar o projeto que precisa, consulte a supervisão.
Selecionar Atividades da Equipe

No campo "Lista das Equipes", selecione quais atividades de qual equipe deseja visualizar.

💡 Exemplo Visual - Lista das Equipes
💡 Dica: Filtro para exibir apenas as atividades relevantes da equipe selecionada.
Selecionar Tarefa de Checklist

No campo "Lista de Tarefas Checklist", selecione o checklist que deseja visualizar.

💡 Exemplo Visual - Lista de Tarefas
Adicionar Checklist

Clique no botão verde "Insere Checklist" para começar a trabalhar com o checklist selecionado.

💡 Exemplo Visual - Botão Checklist
Executar o Checklist

Após expandir os itens do checklist selecionado, siga a ordem sequencial proposta para garantir a execução correta de todas as atividades.

🔸 Seguir Ordem: Execute os itens na sequência numerada apresentada no checklist
🔸 Iniciar Item: Clique no botão verde "Iniciar" para começar um item específico
🔸 Controles Liberados: Após iniciar, os botões "Pausar" e "Finalizar" ficarão disponíveis
🔸 Pausar se Necessário: Use o botão "Pausar" para interromper temporariamente a atividade
🔸 Finalizar Item: Ao concluir a atividade, clique em "Finalizar" no CODA
💡 Exemplo Visual - Execução do Checklist

Tela mostrando o checklist expandido com os botões de controle (Iniciar, Pausar, Finalizar)

📋 Fluxo de Execução:
1️⃣ Selecionar item → 2️⃣ Clicar "Iniciar" → 3️⃣ Executar atividade → 4️⃣ Clicar "Finalizar"
⚠️ Importante: Cada item deve ser finalizado no CODA após sua conclusão para manter o controle e rastreamento adequado do progresso do projeto.
