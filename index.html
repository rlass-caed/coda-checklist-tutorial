TUTORIAL DE UTILIZAÇÃO - CODA

Objetivo

Este tutorial tem como objetivo padronizar o uso do CODA entre todas as equipes, garantindo que todos tenham o mesmo conhecimento sobre cadastro de checklists, manutenção e acesso às informações.

SEÇÃO 1

📋 COMO CADASTRAR UM CHECKLIST

IMPORTANTE: Esta área permite cadastrar tarefas (checklists) necessários para realizar uma atividade existente e já mapeada. Para adicionar atividades novas que ainda não foram mapeadas, procure a supervisão para orientação sobre o processo de mapeamento inicial.
Acessar o Coda - Operação de Dados

Faça login no sistema CODA utilizando suas credenciais.

Entrar na Área "Operação de Dados"

Na lista de documentos, clique em "Operação de Dados" para acessar a área de trabalho principal.

Exemplo Visual - Acessar Operação de Dados

Clique no documento "Operação de Dados" na lista de documentos disponíveis.

Navegar até Templates

No painel de navegação esquerdo, siga o caminho hierárquico:

🌍 Supervisão → Bases Compartilhadas → Checklist → Templates

Esta é a área principal para gerenciamento de todos os modelos de checklist, onde você poderá visualizar, criar e modificar templates existentes.

Exemplo Visual - Navegação

A imagem mostra como navegar pela estrutura de pastas até chegar na seção Templates, destacando cada etapa do caminho.

Procurar o Template da Equipe

Na seção de Templates, selecione a equipe e qual atividade deseja inserir um checklist.

Exemplo Visual - Templates por Equipe
Entendendo a Estrutura do Checklist
Como funciona a organização?
Ao criar checklists, eles seguem uma hierarquia em 3 níveis: Atividade > Tópicos > Tarefas.

Para ficar mais claro, pense na Atividade "Fazer café":
  • Atividade: É o objetivo principal (Ex: Fazer café).
  • Tópicos: São as grandes etapas para atingir o objetivo (Ex: Esquentar água e Servir café).
  • Tarefas: São as ações curtas e práticas de cada etapa (Ex: Pegar caneco, Ligar o fogo, Botar na xícara).
Mantenha essa estrutura para facilitar a organização e a correta execução do checklist pelas equipes.
Exemplo Visual - Templates por Equipe
Adicionar Nova Tarefa
  • 🔸 Selecione uma equipe: Ao selecionar a equipe e a respectiva atividade na qual deseja cadastrar uma tarefa, o botão verde adicionará uma nova linha à atividade. Em seguida, será possível editá-la de acordo com o tópico ao qual pertence. Caso a atividade ainda não exista, ela será criada automaticamente.
  • 🔸 Expandir Seção: Clique para expandir e visualizar as atividades da equipe selecionada
  • 🔸 Selecionar Atividade: Escolha a atividade desejada e expanda para ver seus tópicos e tarefas
  • 🔸 Adicionar Nova Tarefa: Defina o tópico que melhor se encaixa a nova tarefa, ou crie um tópico novo, adicione o título e complete com a descrição detalhada
Exemplo Visual
Reordenar e Excluir Tarefas

Após adicionar tópicos e tarefas, você pode reorganizar a ordem de exibição ou remover itens que não são mais necessários.

  • 🔸 Ordenação Automática: O sistema numera automaticamente as tarefas conforme são adicionadas
  • 🔸 Reordenação Manual: Clique e arraste qualquer tarefa com o mouse para a posição desejada
  • 🔸 Atualização Automática: Os números de ordem se ajustam automaticamente após o movimento
  • 🔸 Exclusão de Atividades: Clique com o botão direito na linha e escolha a opção "delete row everywhere"
  • 🔸 Confirmar Exclusão: Confirme a ação quando solicitado pelo sistema
Exemplo Visual - Reordenação e Exclusão

Vídeo demonstrando como reordenar atividades arrastando com o mouse e como excluir itens usando o menu contextual

SEÇÃO 2

MANUTENÇÃO DO CHECKLIST

Realizar Manutenção de Checklists Existentes

Para realizar manutenção em checklists já cadastrados, siga os mesmos passos da SEÇÃO 1 - COMO CADASTRAR UM CHECKLIST até chegar no template. A partir desse ponto, siga os passos específicos de manutenção abaixo:

  • 🔸 Localizar Equipe: Encontre a equipe onde deseja fazer a manutenção
  • 🔸 Expandir Seção: Clique para expandir e visualizar as atividades da equipe selecionada
  • 🔸 Selecionar Atividade: Escolha a atividade desejada e expanda para ver seus tópicos e tarefas
  • 🔸 Realizar Manutenção: Adicionar, editar ou excluir tarefas e tópicos conforme necessário
Exemplo Visual - Processo de Manutenção
Dica: Para adicionar novos itens, use os mesmos procedimentos explicados na Seção 1. Para editar, clique diretamente no campo desejado. Para excluir, use o botão direito na linha e selecione "delete row everywhere".
⚠️ Importante: Lembre-se que os TÓPICOS funcionam como AGRUPADORES de tarefas relacionadas. Mantenha essa organização durante a manutenção.
SEÇÃO 3

COMO UTILIZAR O CHECKLIST

Entrar na Área "Operação de Dados"

Na lista de documentos, clique em "Operação de Dados" para acessar a área de trabalho principal.

Exemplo Visual - Acessar Operação de Dados

Clique no documento "Operação de Dados" na lista de documentos disponíveis.

Escolher Sua Equipe

Na tela inicial, selecione sua equipe na lista de equipes disponíveis.

Nesta tela você pode ver como selecionar sua equipe na lista disponível.

Acessar Painel da Equipe

Entre no painel da sua equipe onde você poderá visualizar todos os projetos cadastrados.

Exemplo Visual - Painel da Equipe

Nesta tela você seleciona o painel da equipe que deseja

Selecionar Projeto

Escolha o projeto desejado e clique no botão verde "Abrir" para acessá-lo.

Exemplo Visual - Seleção de Projeto
Dica de Pesquisa: É possível pesquisar pelo código do projeto usando o campo de pesquisa localizado na parte direita da tela. Isso facilita encontrar projetos específicos em listas extensas.

Selecionar Atividades da Equipe

No campo "Lista das Equipes", selecione a equipe na qual deseja ver as atividades.

Exemplo Visual - Lista das Equipes

Selecionar Tarefa de Checklist

No campo "Lista de Tarefas Checklist", selecione a atividade na qual deseja ver o checklist.

Exemplo Visual - Lista de Tarefas
Adicionar Checklist

Clique no botão verde "Insere Checklist" para começar a trabalhar com o checklist selecionado.

Exemplo Visual - Botão Checklist
Executar o Checklist

Após expandir os itens do checklist selecionado, siga a ordem sequencial proposta para garantir a execução correta de todas as tarefas.

  • 🔸 Seguir Ordem: Execute as tarefas na sequência numerada apresentada no checklist
  • 🔸 Iniciar Item: Clique no botão verde "Iniciar" para começar uma tarefa específica
  • 🔸 Controles Liberados: Após iniciar, os botões "Pausar" e "Finalizar" ficarão disponíveis
  • 🔸 Pausar se Necessário: Use o botão "Pausar" para interromper temporariamente a tarefa
  • 🔸 Finalizar Tarefa: Ao concluir a tarefa, clique em "Finalizar" no CODA
  • 🔸 Campo de Observação: Voce pode adicionar observações a respeito da tarefa que está sendo realizada
Exemplo Visual - Execução do Checklist
Fluxo de Execução:
1️⃣ Selecionar item → 2️⃣ Clicar "Iniciar" → 3️⃣ Executar atividade → 4️⃣ Clicar "Finalizar"
Importante: Cada tarefa deve ser finalizada no CODA após sua conclusão para manter o controle e rastreamento adequado do progresso do projeto.
